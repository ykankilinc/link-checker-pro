# Link Checker Pro - Premium Link Validator

Link Checker Pro is a high-performance Chrome extension designed for SEO specialists and web developers to identify broken links (404s), redirects (301/302), and server errors in real-time.

![Link Checker Pro Logo](icons/icon128.png)

## 🚀 Key Features
- **Real-Time Analysis:** Scans the active page and highlights every link status instantly.
- **Background Persistence:** Keeps scanning even if you close the extension popup. Reopen to see the latest stats.
- **Visual Badges:** Adds status codes (200, 404, etc.) directly next to links on the page for easy identification.
- **CSV Export:** Download full reports with one click for further SEO analysis and documentation.
- **Exclusion Lists:** Filter out specific domains or patterns to focus on exactly what you need.

## 📖 How to Use
1. **Clone** this repository or download the source code.
2. **Go to** `chrome://extensions/` in your Chrome browser.
3. **Enable** "Developer mode".
4. **Click** "Load unpacked" and select the `link-checker-pro` directory.
5. **Open** the extension from your toolbar and click "Start Scan".

## 🛡️ Privacy & Security
We do not collect or store any personal data. All link checking happens within your browser environment. Your privacy is our priority.
